# user-management
User management application using react and node js

There are 2 parent folder containing the application code:

  **node:** 
  Contains code related to back-end of the application (Node.js, express, MongoDB)
  
  _.env_ file inside the root directory contains the below configuration values:
  
  **PORT:** The port on which the backend application runs
  
  **CLIENT_URI:** The mongo DB clusture URL for connecting to the DataBase
  
  **react:** 
  Contains code related to the front end of the application (react.js)
  
  _.env_ file inside the root directory contains the below configuration values:
  
  **API_BASE_URL:** The base URL for connecting to the back end API
